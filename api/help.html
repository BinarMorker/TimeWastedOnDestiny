Wasted on Destiny API - Help

Show Help

Get the help file (current page)
Method : GET URL : api/?help

Parameters

None

Show Version

Get this instance's API version and the currently online version
Method : GET URL : api/?version

Parameters

&fmt (optional)
Format the output to be readable (not for production)

Get Time Wasted

Get a player's information for time wasted, all characters included
Method : GET URL : api/?console=[console]&user=[username]

Parameters

[console]
1 for Xbox, 2 for Playstation
[username]
The player's username
&fmt (optional)
Format the output to be readable (not for production)
Note : Unique results will only refresh once per hour.

Get Leaderboard

Get the leaderboard
Method : GET URL : api/?leaderboard&page=[page]

Parameters

[page]
The leaderboard page
&fmt (optional)
Format the output to be readable (not for production)
Note : There is only 10 results per page.

Get Clan Times

Get a clan's leaderboard
Method : GET URL : api/?clan=[id]&page=[page]

Parameters

[id]
The clan identifier
[page]
The member page
&fmt (optional)
Format the output to be readable (not for production)
Note : Unique results will only refresh once per hour.
Note : There is only 10 results per page.
